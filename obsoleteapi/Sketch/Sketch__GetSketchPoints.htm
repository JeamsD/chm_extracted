Sketch::GetSketchPoints

This method is obsolete and has been superseded by Sketch::GetSketchPoints2.

Description

This method returns an array of SketchPoint objects that exist in this sketch.

 

Syntax (OLE Automation)

retval = Sketch.GetSketchPoints ()

 

Return:

(VARIANT) retval

VARIANT of type SafeArray of Dispatch pointers, which are the SketchPoint objects in this sketch

 

 

Syntax (COM)

See Sketch::IEnumSketchPoints
