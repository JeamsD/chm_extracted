PropertyManagerPage2Handler4::OnNumberboxChanged

This method is obsolete and has been superseded by PropertyManagerPage2Handler5::OnNumberboxChanged.

Description

This method is called when a user changes the value in a number box on this PropertyManager.

 

Syntax (OLE Automation)

void = PropertyManagerPage2Handler4.OnNumberboxChanged ( Id, Value )

 

Input:

(long) Id

ID of number box

Input:

(double) Value

Value

 

Syntax (COM)

status = PropertyManagerPage2Handler4->OnNumberboxChanged ( Id, Value )

 

Input:

(long) Id

ID of number box

Input:

(double) Value

Value

Return:

(HRESULT) status

S_OK if successful
