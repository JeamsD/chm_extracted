ModelDoc2::GetCustomInfoCount

This method is obsolete and has been superseded by ModelDoc2::GetCustomInfoCount2.

Description

This method returns the number of custom info fields that have been defined for the document.

 

Syntax (OLE Automation)

Count = ModelDoc2.GetCustomInfoCount( )

 

Return:

(long) Count

Number of custom info fields

 

Syntax (COM)

status = ModelDoc2->GetCustomInfoCount ( &Count )

 

Output:

(long) Count

Number of custom info fields

Return:

(HRESULT) status

S_OK if successful
