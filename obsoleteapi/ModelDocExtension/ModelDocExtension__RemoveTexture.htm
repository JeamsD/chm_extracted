ModelDocExtension::RemoveTexture

 

This method is obsolete and has been superseded by ModelDocExtension::RemoveTexture2.

Description

This method removes texture from all selected objects.

 

 

Syntax (OLE Automation)

status = ModelDocExtension.RemoveTexture ()

 

Output:

(VARIANT_BOOL) status

TRUE if texture is removed, FALSE if not

 

Syntax (COM)

status = ModelDocExtension->RemoveTexture ( &status)

 

Output:

(VARIANT_BOOL) status

TRUE if texture is removed, FALSE if not

Return:

(HRESULT) status

S_OK if successful
