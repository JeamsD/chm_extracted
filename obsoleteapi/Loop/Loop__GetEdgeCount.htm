Loop::GetEdgeCount

This method is obsolete and has been superseded by Loop2::GetEdgeCount.

 

Description

This method returns the number of edges in the loop.

 

Syntax (OLE Automation)

retval = Loop.GetEdgeCount ()

 

Return:

(long) retval

Number of edges in the loop

 

Syntax (COM)

status = Loop->GetEdgeCount ( &retval )

 

Output:

(long) retval

Number of edges in the loop

Return:

(HRESULT)status

S_OK if successful
