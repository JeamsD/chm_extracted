DrawingDoc::CreateDetailView

This method is obsolete and has been superseded by DrawingDoc::CreateDetailViewAt.

Description

This method creates a detail view using the selected closed spline sketched on a drawing view.

 

Syntax (OLE Automation)

void DrawingDoc.CreateDetailView ()

 

Syntax (COM)

status = DrawingDoc->CreateDetailView ( )

 

Return:

(HRESULT)status

S_OK if successful
