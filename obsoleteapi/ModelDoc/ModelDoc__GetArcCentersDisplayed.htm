ModelDoc::GetArcCentersDisplayed

This method is obsolete and has been superseded by ModelDoc2::GetArcCentersDisplayed.

 

Description

This method gets the display setting for the arc centers.

 

Syntax (OLE Automation)

retval = ModelDoc.GetArcCentersDisplayed ()

 

Return:

(BOOL) retval

TRUE if arc centers are currently displayed, FALSE otherwise

 

Syntax (COM)

status = ModelDoc->GetArcCentersDisplayed ( &retval )

 

Output:

(VARIANT_BOOL) retval

TRUE if arc centers are currently displayed, FALSE otherwise

Return:

(HRESULT) status

S_OK if successful
