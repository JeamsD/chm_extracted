ModelDoc::GetCustomInfoCount

This method is obsolete and has been superseded by ModelDoc::GetCustomInfoCount2.

 

Description

This method returns the number of custom information fields that have been defined for the document.

 

Syntax (OLE Automation)

Count = ModelDoc.GetCustomInfoCount( )

 

Return:

(long) Count

Number of custom information fields

 

Syntax (COM)

status = ModelDoc->GetCustomInfoCount ( &Count )

 

Output:

(long) Count

Number of custom information fields

Return:

(HRESULT) status

S_OK if successful
