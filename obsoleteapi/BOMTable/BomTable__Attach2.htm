BomTable::Attach2

This method is obsolete and has been superseded by BomTable::Attach3.

Description

This method activates the BOM.

 

Syntax (OLE Automation)

retval = BomTable.Attach2 ()

 

Return:

(BOOL) retval

TRUE if it was successfully attached, FALSE if it was not

 

Syntax (COM)

status = BomTable->Attach2 ( &retval )

 

Output:

(VARIANT_BOOL) retval

TRUE if it was successfully attached, FALSE if it was not

Return:

(HRESULT)status

S_OK if successful

 

Remarks

Call this method before you use any of the BomTable methods.
