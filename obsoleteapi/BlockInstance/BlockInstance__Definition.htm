BlockInstance::Definition

This property is obsolete and has been superseded by SketchBlockInstance::Definition.

Description

This property gets the block definition of this block.

 

Syntax (OLE Automation)

Definition = BlockInstance.Definition (VB Get property)

 

Definition = BlockInstance.GetDefinition ( ) (C++ Get property)

 

Property:

(LPBLOCKDEFINITION) Definition

Block definition

 

Syntax (COM)

status = BlockInstance->get_Definition ( &Definition )

 

Property:

(LPBLOCKDEFINITION) Definition

Block definition

Return:

(HRESULT) status

S_OK if successful
