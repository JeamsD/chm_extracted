BlockInstance::GetAnnotation

This method is obsolete and was not superseded.

Description

This method gets the general annotation object for this specific annotation.

 

Syntax (OLE Automation)

retval = BlockInstance.GetAnnotation ( )

 

Output:

(LPANNOTATION) retval

Pointer to the Annotation object

 

Syntax (COM)

status = BlockInstance->GetAnnotation ( &retval )

 

Output:

(LPANNOTATION) retval

Pointer to the Annotation object

Return:

(HRESULT) status

S_OK if successful
