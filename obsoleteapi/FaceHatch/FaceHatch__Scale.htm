FaceHatch::Scale

 

This property is obsolete and has been superseded by FaceHatch::Scale2.

Description

This property gets the scale for the face hatch.

 

Syntax (OLE Automation)

retVal = FaceHatch.Scale (VB Get property)

 

retVal = FaceHatch.GetScale ( ) (C++ Get property)

Property:

(double) retVal

Scale for the face hatch

Syntax (COM)

status = FaceHatch->get_Scale ( &retVal )

 

Property:

(double) retVal

Scale for the face hatch

Return:

(HRESULT) status

S_OK if successful
