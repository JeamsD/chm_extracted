PropertyManagerPage2Handler3 Object

This object is obsolete and has been superseded by PropertyManagerPage2Handler4.

 

Must be implemented by the add-in application to handle callbacks from PropertyManagerPage2.

NOTE: To use this interface in a VBA macro, you must reference the SolidWorks exposed type libraries for add-in use type library specific to the version of SolidWorks that you are running.

For all unimplemented methods of this object, the add-in application can return E_NOTIMPL.
