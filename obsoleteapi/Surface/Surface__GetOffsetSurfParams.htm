Surface::GetOffsetSurfParams

This method is obsolete and has been superseded by Surface::GetOffsetSurfParams2.

Description

This method gets the distance value of this offset surface.

Syntax (OLE Automation)

retval = Surface.GetOffsetSurfParams ()

 

Return:

(double) retval

Offset value for the surface

 

Syntax (COM)

status = Surface->GetOffsetSurfParams ( &retval )

 

Output:

(double) retval

Offset value for the surface

Return:

(HRESULT) status

S_OK if successful
