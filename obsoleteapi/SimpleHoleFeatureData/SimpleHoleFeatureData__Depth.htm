SimpleHoleFeatureData::Depth

This property is obsolete and has been superseded by SimpleHoleFeatureData2::Depth.

 

Description

This property gets or sets the simple hole feature hole depth.

 

Syntax (OLE Automation)

depth = SimpleHoleFeatureData.Depth  (VB Get property)

SimpleHoleFeatureData.Depth = depth  (VB Set property)

 

depth = SimpleHoleFeatureData.GetDepth ( )  (C++ Get property)

SimpleHoleFeatureData.SetDepth ( depth )  (C++ Set property)

 

 

Property:

(double) depth

Hole depth

 

 

Syntax (COM)

status = SimpleHoleFeatureData ->get_Depth ( &depth )

status = SimpleHoleFeatureData ->put_Depth ( depth )

 

Property:

(double) depth

Hole depth

Return:

(HRESULT) status

S_OK if successful
