BlockDefinition Object

This method is obsolete and has been superseded by SketchBlockDefinition.

 

 

 

 

 

The BlockDefinition interface allows access to information about a block definition. Using this interface, you can:

  • Get and set:

    • Name of the block

    • Whether the block links to an external file, and get or set the name of that file

  • Get:

    • All instances and number of blocks

    • Sketch of this block definition

    • All notes and dimensions and the number of all notes and dimensions

 

Use the Accessors link to get a list of functions that return this object.
