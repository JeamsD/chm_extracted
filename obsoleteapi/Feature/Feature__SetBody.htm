Feature::SetBody

This method is obsolete and has been superseded by Feature::SetBody2.

Description

This method replaces an imported base feature body.

 

Syntax (OLE Automation)

retval = Feature.SetBody ( bodyIn)

 

Input:

(LPDISPATCH) bodyIn

Dispatch pointer to the temporary body that replaces the base feature body

Return:

(BOOL) retval

TRUE if original body was successfully replaced, FALSE if not

 

Syntax (COM)

status = Feature->ISetBody ( bodyIn, &retval )

 

Input:

(LPBODY) bodyIn

Pointer to the temporary body that replaces the base feature body

Output:

(VARIANT_BOOL) retval

TRUE if original body was successfully replaced, FALSE if not

Return:

(HRESULT)status

S_OK if successful

 

Remarks

The replacement body must be created as an imported body using Body2::CreateBodyFromSurfaces or Body::CreateTempBodyFromSurfaces, or as a copy of an imported body generated by the same methods.

If this feature is not the base feature or the body is not an imported body, then the method returns FALSE.

This method replaces the base body or reference surface body if you created either of these features from imported geometry. If child features exist, then you can use Face2::GetFaceId to get surface IDs from the old body so that you can set them on the replacement body with Face2::SetFaceId. This mechanism enables child features to re-attach themselves to the replacement body because this method cannot do this automatically.
