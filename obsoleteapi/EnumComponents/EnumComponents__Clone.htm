EnumComponents::Clone

This method is obsolete and has been superseded by EnumComponents2::Clone.

Description

This method clones the enumerator.

 

Syntax (OLE Automation)

Not available.

 

 

Syntax (COM)

status = EnumComponents->Clone ( ppenum )

 

Output:

(LPENUMCOMPONENTS*) ppenum

Pointer to the cloned list of component objects

Return:

(HRESULT) status

S_OK if successful
