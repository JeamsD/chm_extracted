Face::GetLoopCount

This method is obsolete and has been superseded by Face2::GetLoopCount.

Description

This method returns the total number of loops in the face.

 

Syntax (OLE Automation)

retval = Face.GetLoopCount ()

 

Return:

(long) retval

Number of loops in this face

 

Syntax (COM)

status = Face->GetLoopCount ( &retval )

 

Output:

(long) retval

Number of loops in this face

Return:

(HRESULT) status

S_OK if successful
