Face::GetArea

This method is obsolete and has been superseded by Face2::GetArea.

Description

This method returns the area of the face.

 

Syntax (OLE Automation)

retval = Face.GetArea ()

 

Return:

(double) retval

Face area in square meters

 

Syntax (COM)

status = Face->GetArea ( &retval )

 

Output:

(double) retval

Face area in square meters

Return:

(HRESULT) status

S_OK if successful
