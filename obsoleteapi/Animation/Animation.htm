Animation Object

This interface is:

  • obsolete and has not been superseded.

  • nonfunctional in SolidWorks 2008 and later.

    Use the interfaces related to motion studies introduced in SolidWorks 2008 to access animation and simulation.

 

 

 

 

Allows access to an animation and the Animation Controller pop-up toolbar, which is currently only available when replaying a Physical Simulation in the SolidWorks API.

NOTE: Do not confuse the Animation Controller pop-up toolbar available in the core SolidWorks software with SolidWorks Animator, a SolidWorks add-in. SolidWorks Animator provides more functionality than the Animation Controller pop-up toolbar. See the SolidWorks Animator online Help for details.

Use the Accessors link to obtain a list of functions that return this object.
