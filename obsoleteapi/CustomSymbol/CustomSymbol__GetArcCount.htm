CustomSymbol::GetArcCount

This method is obsolete and has been superseded by Note::GetArcCount.

Description

This method gets the number of arc items in this custom symbol.

 

Syntax (OLE Automation)

retval = CustomSymbol.GetArcCount ()

 

Return:

(long) retval

Number of arcs

 

Syntax (COM)

status = CustomSymbol->GetArcCount ( &retval )

 

Output:

(long) retval

Number of arcs

Return:

(HRESULT)status

S_OK if successful
