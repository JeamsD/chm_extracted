Remarks

This method is valid only if ICWRemoteLoad::AllowDistributedCoupling returns false (i.e., for frequency, buckling, nonlinear dynamic, or linear dynamic studies). If ICWRemoteLoad::AllowDistributedCoupling returns true (i.e., for linear static, topology, or nonlinear static studies), use ICWRemoteLoad::SetMomentOrRotationValuesEx instead of this method.
