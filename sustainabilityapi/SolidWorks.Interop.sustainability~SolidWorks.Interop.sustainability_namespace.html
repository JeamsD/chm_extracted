Sustainability

Interfaces

InterfaceDescription
ISustainabilityApp

Allows access to all of the other interfaces exposed in the Sustainability API.

ISustainabilityAssemblyProcess

Allows access to the Sustainability assembly process data for the current assembly.

ISustainabilityAssemblyUse

Allows access to the Sustainability use data for the current assembly.

ISustainabilityEndOfLife

Allows access to the Sustainability disposal data for the current part or assembly.

ISustainabilityEnvironmentalImpact

Allows access to the Sustainability environmental impact of the design of the current part or assembly.

ISustainabilityManufacturing

Allows access to the Sustainability manufacturing data for the current part.

ISustainabilityMaterial

Provides the ability to assign materials to the current part or the components of the current assembly.

ISustainabilityTransportation

Allows access to the Sustainability transportation data for the current part or assembly.

ISustainabilityUse

Allows access to the Sustainability use data for the current part.
