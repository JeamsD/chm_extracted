Get DimXpert Extrude Feature Example (VB.NET)

This example builds a part to demonstrate usage of the SOLIDWORKS Document Manager API for a DimXpert extrude feature.

'----------------------------------------------------------------------------
' 1. Open public_documents\samples\tutorial\api\plate_ads_plusminus.sldprt.
' 2. Open the DimXpert toolbar from View > Toolbars
'    (select the first instance of Toolbars on the View menu).
' 3. Create the Extrude Feature:
'    a. Click Auto Dimension Scheme on the DimXpert toolbar.
'    b. Ensure that all feature filters are selected.
'    c. Click the green check mark to accept the settings.
' 4. Observe the following DimXpert features on the
'    DimXpertManager tab:  Hole Pattern1 and Pocket1.
' 5. Close the part, saving it to another name.
'    NOTE: Because this part is used in a SOLIDWORKS online tutorial,
'          do not save any changes when you close the original file name.
' 6. Read how to load and run Code_Example_VBNET.
'----------------------------------------------------------------------------
