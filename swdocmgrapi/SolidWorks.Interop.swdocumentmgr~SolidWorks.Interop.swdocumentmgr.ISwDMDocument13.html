Remarks

Before you can access this object, you must call ISwDMApplication::GetDocument.

NOTES:

  • You can access an ISwDMDocument13 object by calling QueryInterface on an ISwDMDocument object in C++ or by direct assignment in Visual Basic.

  • The SwDMDocument object can be assigned to an ISwDMDocument13 variable, just like the SOLIDWORKS IModelDoc2 object can be assigned to a SOLIDWORKS IPartDoc variable.
