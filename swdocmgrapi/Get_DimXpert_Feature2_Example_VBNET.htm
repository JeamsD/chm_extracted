SOLIDWORKS Document Manager API Help

Get More DimXpert Feature Examples (VB.NET)

This example builds a part to demonstrate usage of the SOLIDWORKS Document Manager API for the following DimXpert features:

    *  Compound hole

    *  Compound notch

'----------------------------------------------------------------------------

' 1. Open public_documents\samples\tutorial\api\cover_with_dimensions.sldprt.

' 2. Open the DimXpert toolbar from View > Toolbars

'    (select the first instance of Toolbars on the View menu).

' 3. Observe the following DimXpert features on the

'    DimXpertManager tab:  Simple Hole2 and Notch1.

' 4. Close the part, saving it to another name.

'    NOTE: Because this part is used in a SOLIDWORKS

'          online tutorial, do not save any changes to the original file name.

' 5. Read how to load and run Code_Example_VBNET with this part.

'---------------------------------------------------------------------------
