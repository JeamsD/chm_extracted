Remarks

This method only returns data for documents created using SOLIDWORKS 2004 (Version 2500) and later. To get the version of a document, use ISwDMDocument::GetVersion.

The 13 elements in the array are:

Element

Description

[0 2]:

(x, y, z) Center of mass (m)

[3]:

Volume (m^3)

[4]:

Surface area (m^2)

[5] :

Mass (kg)

[6]:

momXX (kg*m^2)

[7] :

momYY (kg*m^2)

[8] :

momZZ (kg*m^2)

[9] :

momXY (kg*m^2)

[10]:

momZX (kg*m^2)

[11]:

momYZ (kg*m^2)

[12]:

Include hidden
