Get DimXpert Sphere and Datum Example (VB.NET)

This example builds a part to demonstrate usage of the SOLIDWORKS Document Manager API for the following DimXpert entities:

* Sphere feature

* Datum annotation

'----------------------------------------------------------------------------

' 1. Open public_documents\samples\tutorial\floxpress\ball valve\Ball.sldprt.

' 2. Open the DimXpert toolbar from View > Toolbars (select the first

'     instance of Toolbars on the View menu).

' 3. Click Datum on the DimXpert toolbar.

' 4. Click the ball of the part.

' 5. Click outside the part to place the datum annotation.

' 6. Click the green check mark to accept the new datum feature.

' 7. Observe the following DimXpert feature on the DimXpertManager

'    tab:  Sphere1.

' 8. Close the part, saving it to another name.

'    NOTE:  Because this part is used in a SOLIDWORKS online tutorial,

'           do not save any changes when you close the original file name.

' 9. Read how to load and run Code_Example_VBNET with this part.

'----------------------------------------------------------------------------
