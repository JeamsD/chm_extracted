The SOLIDWORKS Costing tool helps designers and manufacturers:

  • calculate these manufacturing methods by automating the cost estimation and quotation process:
    • 3D printing
    • Casting
    • Machining
    • Plastic
    • Sheet metal
    • Structural
  • make decisions based on the cost to manufacture.
  • create quotes for customers. Whenever a design is changed, the new updated cost can be seen immediately, along with a detailed cost breakdown. Automatic cost reports can also be generated. 

You can further automate the cost analysis process using the SOLIDWORKS Costing Application Programming Interface (API), which is an OLE programming interface to SOLIDWORKS Costing functionality. The API contains functions that provide direct access to the SOLIDWORKS Costing environment that you can call from Visual Basic for Applications (VBA), VB.NET, C++ 6.0, C++, and C#. It is assumed that you are familiar with at least one of these programming languages.

This online reference manual documents the interfaces available in the SOLIDWORKS Costing API, including their associated methods and properties. SOLIDWORKS Costing and its API are only available in SOLIDWORKS Professional and SOLIDWORKS Premium.
