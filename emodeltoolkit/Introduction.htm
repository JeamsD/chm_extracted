The eModelToolkit is:

  • a library for creating eDrawings part (.eprt), assembly (.easm), and drawing (.edrw) files.

  • implemented as a C++ DLL that exports a set of classes, functions, and enumerations to which your application can link.

EModelToolkit.dll is not a COM server DLL; therefore, you must be familiar with C++ to use this toolkit. Familiarity with OpenGL®, HOOPS®, or any other tessellation-based graphics system will also be helpful.
