Document Structure Classes

See Also

Class

Description

eDAssembly

Contains one or more parts (eDPart) and assemblies (eDAssembly).

eDBody

Contains shells (eDShellList), curves (eDCurveList), and marker points (eDMarkerList).

eDCamera

Represents a camera.

eDConfiguration

Contains either a root-level assembly (eDAssembly) or a root-level part (eDPart).

eDDrawing

Contains one or more sheets.

eDPart

Contains one or more bodies (eDBody).

eDScene

Represents an eDrawings document (.eprt, .easm, or .edrw file) and contains:

  • either one or more configurations (eDConfiguration) or a drawing (eDDrawing).

  • the methods to create bodies, textures, and the eDSerializer object, which allows the user to save the scene to disk as a .eprt, .easm, or .edrw file.

eDSerializer

Saves a scene (eDScene) to disk.

eDSerializerParams

Contains the options for saving eDrawings documents to disk.

eDSheet

Contains one or more views.

eDTime

Contains the date and time that the file will expire or not open.

eDView

Contains part, assembly, and 2D graphical data.
