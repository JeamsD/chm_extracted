Class

Description

eDAnimation

Contains either an animation sequence or a set of explode steps.

eDCurveList

Contains one or more circles, arcs, ellipses, elliptical arcs, lines, polylines, individual polygons, and splines.

eDFacets

Represents a set of planar polygonal facets.

eDFont

Represents the font.

eDHoleProperties

Contains hole axis and diameter data.

eDMarkerList

Contains one or more 3D point markers.

eDMassProperties

Contains mass, volume, surface area, and center of gravity data.

eDShellList

Contains one or more shells. A shell is a set of planar polygonal facets.
