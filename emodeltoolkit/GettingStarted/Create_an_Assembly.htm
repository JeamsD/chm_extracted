Create an Assembly

See Also

You will typically follow these steps to create an assembly using the eModelToolkit.

  1. Create an application using eDApplication::Create.

  2. Create a scene (eDScene) using eDApplication::CreateScene.

  3. Create and add a new configuration (eDConfiguration) to that scene using eDScene::CreateConfiguration.

  4. Create and add a new assembly (eDAssembly) to that configuration using eDConfiguration::CreateAssembly. This is the top-level root component.

  5. To create a new component with no children:

    1. Create and add a new part (eDPart) to the top-level root component using eDAssembly::CreatePart.

    2. Create a new body (eDBody) and populate it with data using eDScene::CreateBody and eDBody methods.

    3. Add that body to the part using eDPart::AddBody.

  6. To create a subassembly, create a new assembly (eDAssembly) using eDAssembly::CreateSubAssembly. The parent-child link is created for you.

Add parts or more subassemblies to that assembly using eDAssembly::CreatePart and eDAssembly::CreateAssembly, respectively.

  1. Repeat Steps 5 and 6 to create additional components.

  2. Repeat Steps 3 - 6 to create additional configurations.

  3. Create the serialization object (eDSerializer) from eDScene::CreateSerializer.

  4. Save the scene to disk using eDSerializer::Serialize with eDSerializerParams as a parameter.
