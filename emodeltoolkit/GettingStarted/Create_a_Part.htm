Create a Part

See Also

You will typically follow these steps to create a part using the eModelToolkit.

  1. Create an application using eDApplication::Create.

  2. Create a scene (eDScene) using eDApplication::CreateScene.

  3. Create and add a new configuration (eDConfiguration) to that scene using eDScene::CreateConfiguration.

  4. Create and add a new part (eDPart) to that configuration using eDConfiguration::CreatePart.

  5. Create a new body (eDBody) and populate it with data using eDScene::CreateBody and eDBody methods.

NOTE: eDCurveList, eDShellList, and eDMarkerList are all independent. In this toolkit, you do not bound curves with marker points or bound faces with curves. Shells may or may not combine to form recognizable surfaces or solid bodies.

  1. eDBody::CreateCurveList //Create sketch segment

    1. eDCurveList::InsertSpline

    2. eDCurveList::InsertLine and so on.

  2. eDBody::CreateShellList

eDShellList::InsertFacets //Input face geometry data

  1. eDBody::CreateMarkerList

eDMarkerList::InsertMarker //Input point data

  1. Add the body to the part with eDPart::AddBody.

  2. Repeat Steps 3 - 6 to add additional configurations.

  3. Create the serialization object (eDSerializer) using eDScene::CreateSerializer.

  4. Save the scene to disk using eDSerializer::Serialize with eDSerializerParams as a parameter.
