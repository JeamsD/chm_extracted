eDView Class

Example

Accessors

Availability

Contains part, assembly, and 2D data and are contained within eDSheet objects.

Header file code

class eDView : public eDObject

{

public:

 

eDOutcome CreatePart( const ECHAR *partname, const eDAppearance &appearance, eDPart *& part ); // Creates a part

 

eDOutcome CreateAssembly( const ECHAR *assemblyname, const eDAppearance &appearance, eDAssembly *& assembly ); //Creates an assembly

 

eDOutcome CreateCurveList( const eDAppearance &appearance, eDCurveList *& curvelist ); //Creates one or more curves

 

eDOutcome CreateMarkerList( const eDAppearance &appearance, eDMarkerList *& markerlist ); //Creates one or more 3D marker points

 

eDOutcome SetClipBoundary( int npoints, eDPt3d *pointdata ); //Sets the clip boundary. The polygon generated by this method only clips eDShellList geometry; that is, facet data added through eDBody::CreateShellList and added to a view through eDView::CreatePart.

 

};
