Public Properties

 NameDescription
AttachmentsGets the non-SOLIDWORKS attachments for this SOLIDWORKS document.  
ConfigurationsGets the configurations available for the specified document and revision.  
DateModifiedGets the date and time that this document was last modified.  
DescriptionGets the description of the document.  
ExtReferencesGets the links to external documents that this document references.  
ExtWhereUsedGets the links to external document that reference this document.  
HasPDFGets whether this drawing document has a PDF file in the vault.  
IsTopLevelGets whether this document is at the top-level project.  
NameGets or sets the name of the document.  
NotesGets the notes associated with this document.  
NumberGets the number of this document.  
OwnerGets the current owner of this document, which may be different from the author of the document.  
projectGets the name of the project to which this document belongs.  
PropertiesGets the properties of this document.  
ReferencesGets the links to the documents referenced by this document.  
RevisionGets the revision for this document.  
RevisionLabelsGets the next suggested revisions for the document.  
RevisionsGets all of the revisions of this document, in order from the oldest to newest.  
SizeGets the size (in bytes) of a document smaller than 2GB or the size that represents the 32 low-order bits of a document larger than 2GB.  
SizeHGets the size that represents the 32 high-order bits of a document larger than 2GB.  
SolidWorksVersionGets the SOLIDWORKS version of the document.  
UpdateStampGets the date on which this document was last updated.  
WhereUsedGets the links to the documents that reference this document.
