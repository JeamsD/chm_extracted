SOLIDWORKS Workgroup PDM is a project-data management application designed to run both natively inside of the SOLIDWORKS environment and as a standalone application.

The SOLIDWORKS Workgroup PDM Application Programming Interface (API) is an OLE programming interface to SOLIDWORKS Workgroup PDM, which allows you to automate and customize SOLIDWORKS Workgroup PDM. The API contains functions that you can call from Visual Basic for Applications (VBA), VB.NET, C++ 6.0, C++/CLI, and C#. These functions provide direct access to the SOLIDWORKS Workgroup PDM environment. It is assumed that you are familiar with at least one of these programming languages.

This online reference manual documents the objects and collections available in the SOLIDWORKS Workgroup PDM API, including their associated methods and properties.

You need WinZip installed on your computer to download the sample project. See http://www.winzip.com for details.
