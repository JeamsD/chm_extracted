Accessing Selections that Define Features

To access selections that define features:

  1. First, call the AccessSelections method for that feature data object (e.g., ICosmeticThreadFeatureData, IExtrudeFeatureData2, IRevolveFeatureData2, ISweepFeatureData, etc.).

  2. Then, call the desired method for that feature data object.

  3. Finally, release the feature data object:
