SOLIDWORKS API Help

Default Paths

In many Help topics, italicized strings appear instead of the default paths. This table describes these strings and the default paths to which they correspond.

This string... Is the default path where...
install_dir SOLIDWORKS is typically installed on a local computer: C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS
public_documents Model documents for use with API Help examples are typically installed on a local computer in the default system public folder path: C:\Users\Public\Documents\SOLIDWORKS\SOLIDWORKS 20nn

NOTE: 20nn represents the version of SOLIDWORKS; e.g., 2017.
