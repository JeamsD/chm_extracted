SOLIDWORKS API Help

Helper Functions

The general helper functions, whose header files were previously provided in public_documents\appcomm, have been superseded by the following API interfaces and their associated methods and properties:

  • IMathPoint

  • IMathTransform

  • IMathUtility

  • IMathVector

  • IUserUnit

The old helper functions, which were independent of the SOLIDWORKS API and provided for your convenience, have been phased out and might not work in SOLIDWORKS 2001Plus and later.
