SOLIDWORKS API Help

Arrays

This topic contains some general information about using arrays in SOLIDWORKS.

  • In the Microsoft .NET Framework, arrays derive from Object. See the Print Drawing as High Quality (C#) and Pack and Go an Assembly (C#) for examples.
  • Passing an array of elements of the wrong size to SOLIDWORKS might not do anything and might generate an error. For details about longs vs. integers, see Long vs. Integer.
  • See Unmanaged C++ and C++/CLI Code Differences for details on handling arrays in unmanaged and managed C++.
  • See IDispatch Object Arrays as Input in .NET for details on using the DispatchWrapper object for non-standard objects like SOLIDWORKS interfaces, which are manipulated through the IDispatch interface.
  • See the following topics for details about double arrays and integer pairs:
    • Unpacking in VB.NET and VBA
    • Unpacking in C++
    • Unpacking in C#     
  • See the following topics for details about SafeArrays:
    • Passing SafeArrays in C++
    • Passing SafeArrays in VB
