Remarks

This method is valid only if IThreadFeatureData::EndCondition is set to swEndConditions_e.swEndCondUpToSelection.

Use IModelDocExtension::SelectByRay with Mark = 1 to select the end condition reference.

See Accessing Selections that Define Features for additional details on using this method.
