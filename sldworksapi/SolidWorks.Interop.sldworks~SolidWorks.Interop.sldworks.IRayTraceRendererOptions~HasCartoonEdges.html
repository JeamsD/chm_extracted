Remarks

This property is only available when IRayTraceRendererOptions::ContourCartoonRenderingEnabled is true and IRayTraceRendererOptions::RenderType is swRayTraceRenderingType_e.swRayTraceCartoon.
To set cartoon... Set IRayTraceRendererOptions::HasCartoonEdges to... Set IRayTraceRendererOptions::HasCartoonShading to...
Edges True False
Shading False True
Edges and shading True True
