Remarks

The index for both rows and columns is 0-based.

When you set a column type, the title is automatically changed to match that column type. If you change the column type to be a custom property column, then the column title is empty and you must set the column title using ITableAnnotation::SetColumnTitle.

A BOM table column cannot be changed to a Quantity type column in the SOLIDWORKS user interface. This method is consistent with the user interface and should return a failure; that is, a value < 0.
