Remarks

For example, a component with these configurations:

  • TenInchDiameter

  • ThirteenInchDiameter

  • TwentyInchDiameter

might have a BoundingValues array of [9,11,12,14,19,21], which specifies a +1 tolerance over each configuration parameter.
