Remarks

If Override is:

  • true, then specify Thickness using an override gauge thickness. You should also use ISheetMetalGaugeTableParameters::SetRadius to override the bend radius.
  • false, instead of calling this method use ISheetMetalGaugeTableParameters::GetAvailableGaugeNumbers and ISheetMetalGaugeTableParameters::SetCurrentGaugeNumber to change the current gauge number. Each gauge number has a default thickness.
