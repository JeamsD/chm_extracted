Return Value

  • in-process, unmanaged C++: Pointer to an array containing the hit points (see Remarks)
  • VBA, VB.NET, C#, and C++/CLI: Not supported

    See In-process Methods for details about this type of method.

    • Remarks

    The array returned contains information about each hit of each ray on the given bodies, in the following format:

    For each intersection found: 

    • (double) BodyIndex: index in BodiesIn array of the body that was hit

    • (double) RayIndex: index in VectorsIn array of rays

    • (double) IntersectionType: type of intersection found. See swRayPtsResults_e . Entry/Exit information (swRayPtsResultsENTER and swRayPtsResultsEXIT) are only returned when swRayPtsOptsENTRY_EXIT is specified in the options argument in the IModelDoc2::IRayIntersections call. If more than one intersection type is found, then the IntersectionType contains an OR'd set of values from the swRayPtsResults_e enumeration.

    • (double) x,y,z: coordinates of hit point

    • (double) nx,ny,nz: normal (optionally appears when swRayPtsOptsNORMALS is specified in the Options argument in the IModelDoc2::RayIntersections call).
