Remarks

FileName is used only:

When the handler of this event...

Returns...

FileCloseNotify non-0 value
PromptForFileNameNotify False
ReferencedFilePreNotify False

Because the filename specified is used only when PromptForFileNameNotify and ReferencedFilePreNotify return false, you cannot specify an initial filename for SOLIDWORKS to use in the standard dialog. Instead, you must provide your own file dialog and return the filename after the user selects it.
