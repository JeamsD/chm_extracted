Remarks

To successfully use this method, you must first turn on 3D Interconnect by either:

  • Setting Tools > Options > System Options > Import > Enable 3D Interconnect

    - or -

  • Calling ISldWorks::SetUserPreferenceToggle(swUserPreferenceToggle_e.swMultiCAD_Enable3DInterconnect, True)

The feature is imported and positioned relative to the origin using coordinates of the imported file. To edit the feature, use IFeature::GetImportedFeatureParameters and IFeature::SetImportedFeatureParameters.

See the IImport3DInterconnectData Remarks.
