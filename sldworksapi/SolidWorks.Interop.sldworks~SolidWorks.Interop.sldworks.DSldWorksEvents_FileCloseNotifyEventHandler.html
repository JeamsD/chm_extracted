Example

You must be a SOLIDWORKS subscription customer to access the SOLIDWORKS API forum's documents.

  1. Open https://forum.solidworks.com/community/solidworks/api?view=documents.
  2. Log in.
  3. Click SOLIDWORKS > API to open the SOLIDWORKS API forum.
  4. Click the Documents tab.
  5. Locate and download the Close and Reopen a Drawing Document (VBA, VB.NET, C#).

Remarks

This event occurs after a drawing document is closed using ISldWorks::CloseAndReopen.

If developing a C++ application, use swAppFileCloseNotify to register for this notification.
