SolidWorks.Interop.sldworks Namespace > ISweepFeatureData Interface : GetPathAlignmentDirectionVector Method (ISweepFeatureData)



Type

Type of direction vector as defined in swSelectType_e:

  • swSelDATAMAXES (axis)
  • swSelDATUMPLANES (plane)
  • swSelEDGES (edge)
  • swSelFACES (face)
Gets the direction vector of the specified type for this sweep feature.

  • swSelDATUMPLANES (plane)

Return Value

Visual Basic for Applications (VBA) Syntax

See SweepFeatureData::GetPathAlignmentDirectionVector.

Remarks

This method is valid only if:

  • ISweepFeatureData:TwistControlType is set to swTwistControlType_e.swTwistControlFollowPath.

     - and -

  • ISweepFeatureData::PathAlignmentType is set to swTangencyType_e.swTangencyDirectionVector.

See Accessing Selections that Define Features for additional details.
