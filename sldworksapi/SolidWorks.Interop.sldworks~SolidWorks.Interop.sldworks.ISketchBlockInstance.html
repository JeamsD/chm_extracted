Remarks

Using this interface, you can:

  • get the block instance's block definition

  • get the sketch in which the block instance is present

  • Get or set the block instance's:

    • Angle

    • Attributes (notes that have tag names and that are not read-only)

    • Layer

    • Leader styles

    • Name

    • Position

    • Scale

    • Text display state

  • Get block instances on a drawing sheet format and edit the sheet format using IDrawingDoc::EditTempate and IModelDoc2::GetActiveSketch2 or IModelDoc2::IGetActiveSketch2, which gains access to the sketch that contains the block instances.

See Block Definitions and Block Instances for details.
