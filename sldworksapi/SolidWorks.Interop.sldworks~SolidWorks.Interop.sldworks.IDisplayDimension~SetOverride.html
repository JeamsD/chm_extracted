Visual Basic for Applications (VBA) Syntax

Remarks

This method can only be used on a display dimension in a drawing. In a part or assembly, this method takes no action and returns false.

 

If Override is set to false, then the Value argument is ignored.

 

Use IDisplayDimension::GetOverride to get whether the actual dimension value or another value is displayed.   

 

Use IDisplayDimension::GetOverrideValue to get the value to display instead of the actual dimension value.
