Remarks

If the center mark is in a center mark set (i.e., a linear or circular pattern), then use ICenterMark::GroupCount to get the range of valid values for the GroupID parameter. You can use a value from 0 to ICenterMark::GroupCount for the GroupID parameter. To determine if a center mark is in a center mark set, use ICenterMark::IsGrouped.
