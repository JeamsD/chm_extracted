SolidWorks.Interop.sldworks Namespace > IDrawingDoc Interface : PasteSheet Method (IDrawingDoc)



InsertOption
Option as defined in swInsertOptions_e
RenameOption
Option as defined in swRenameOptions_e; 1 to rename duplicate section, detail or auxiliary view names; 2 to not rename
Copies and pastes a drawing sheet to the specified location of the drawing document, optionally renaming whenever duplicate names occur.
