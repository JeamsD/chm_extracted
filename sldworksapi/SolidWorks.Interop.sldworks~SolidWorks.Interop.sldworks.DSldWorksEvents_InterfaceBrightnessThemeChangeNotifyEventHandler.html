Remarks

The Colors array is indexed by swInterfaceBrightnessColor_e.

 

To get the SOLIDWORKS background:

  • theme, use ISwHtmlInterface::GetInterfaceBrightnessTheme.
  • theme and colors, ISldWorks::GetInterfaceBrightnessThemeColors. 

If developing a C++ application, use swAppInterfaceBrightnessThemeChangeNotify to register for this notification.
