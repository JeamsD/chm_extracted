Remarks

The index value is 0-based. Therefore, a valid index value is greater than or equal to 0, but less than the number of leaders on this annotation. Use IAnnotation::GetArrowHeadCount to find the number of leaders on this annotation. If the index value is invalid, SOLIDWORKS returns the arrowhead style as -1, and returns an S_FALSE status (COM interface).

 

Use IAnnotation::SetArrowHeadSizeAtIndex to set the arrow head size of a specific leader.
