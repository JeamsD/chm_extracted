SOLIDWORKS API Help

System Options > Export > PLY

To display the dialog:

Click Tools > Options > System Options > Export > PLY in File Format.

- or -

  1. Click File > Save As.
  2. In Save as type, select Polygon File Format.
  3. Click Options.
