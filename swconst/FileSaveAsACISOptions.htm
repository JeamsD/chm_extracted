SOLIDWORKS API Help

System Options > Export > ACIS

To display the dialog:

Click Tools > Options > System Options > Export > ACIS in File Format.

- or -

  1. Click File > Save As.
  2. In Save as type, select ACIS.
  3. Click Options.
