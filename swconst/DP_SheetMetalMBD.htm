Bend notes - Format

IModelDocExtension::SetUserPreferenceToggle(swUserPreferenceToggle_e.swSheetMetalMBDFormat, swUserPreferenceOption_e.swDetailingNoOptionSpecified, <Value>)

True value only

This property is write-true only. The default setting keeps the current bend note format.

To exchange the current bend note format to another that is specified in a bend note file:

  1. Specify a new bend note file in System Options > File Locations > Show folders for > Sheet Metal Bend Line Note File > Folders or set it using

    ISldWorks::SetUserPreferenceStringValue(swUserPreferenceStringValue_e.swFileLocationsBendNoteFormatFile, <Value>)

    See System Options > File Locations documentation. 

  2. Set this property to true.
