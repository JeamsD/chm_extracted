IModelDocExtension::GetUserPreferenceToggle(swUserPreferenceToggle_e.swDetailingNoDimSpecificOptionSpecified, swUserPreferenceOption_e.swDetailingNoOptionSpecified)

 Boolean value Specifies whether to apply the specified decimal rounding method to:
  • any numeric display where rounding occurs, including dialogs.
    - or -
  • dimensions on their tolerances, including dimensions whose values are derived from equations.

NOTE: Check box is not available if Round half away from zero is selected.
