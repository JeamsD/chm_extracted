SOLIDWORKS API Help

System Options > Import > STEP/IGES/ACIS

To display the dialog:

Select Tools > Options > System Options > Import > File Format > STEP/IGES/ACIS.

- or -

  1. Select File > Open.
  2. In Files of type, select IGES, ACIS, or STEP AP203/214/242.
  3. Click Options.
