Remarks

You can use this method to define a simulation feature data object that is used by IMotionStudy::CreateFeature to create these simulation features:

  • 3DContact
  • Damper
  • Force
  • Gravity
  • Motor
  • Spring

A motion study must be active when defining and creating a simulation feature. If a motion study is not active, then this method returns null or Nothing.
